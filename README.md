# 🍽 RecipeRecom

A personalized Recipe Recommendation Android App built with Kotlin.

## Features
- 🔐 Google Sign In (Firebase)
- 🔍 Search by ingredient
- 🍽 Filter by cuisine & mood
- ❤ Save favourite recipes
- 👨‍🍳 Recipes from famous chefs
- ⭐ Ratings, calories & difficulty
- 🎬 Animated splash screen

## Tech Stack
- Kotlin + XML
- Firebase Authentication
- Glide Image Loading
- SharedPreferences
- Bottom Navigation + Fragments
